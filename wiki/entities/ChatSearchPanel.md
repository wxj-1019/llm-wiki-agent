**ChatSearchPanel** is a React component in the [[LLMWikiViewer]] frontend that provides a dual-mode (Wiki/Web) search panel integrated into the [[ChatPage]] sidebar. It features debounced search, per-mode search history persisted in [[LocalStorage]], and quote-to-chat functionality on each result.
