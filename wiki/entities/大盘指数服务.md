# 大盘指数服务

大盘指数服务是[[股票数据服务]]的子模块，提供 A 股 8 只主要指数的实时数据（上证指数、深证成指、创业板指、科创50、沪深300、中证500、中证1000、创业板50）。采用数据库缓存模式，后台每分钟自动刷新入库（仅交易时间9:30-11:30,13:00-15:00），前端 API <50ms 响应。数据源降级链：`rt_idx_k`（Tushare付费）→ 新浪（akshare）→ 东方财富（akshare）。
