# LLMWikiViewer

`LLMWikiViewer` is a full-featured, progressive web application (PWA) frontend for browsing, searching, and interacting with a personal knowledge wiki through both traditional navigation and AI-powered conversational interfaces. Built with React, TypeScript, and a component-based architecture, it serves as the primary user interface for a wiki system that stores pages as Markdown files and exposes them via a structured API. The application is organized around a three-column layout (collapsible sidebars, main content area, and contextual panels) and provides multiple navigation modes: a command palette, keyboard shortcuts, a search page with local Fuse.js and backend unified search, and a dedicated chat page with dual-mode interaction (free-form LLM chat and knowledge-grounded generation). `LLMWikiViewer` also manages file ingestion (upload, paste, URL fetch), streaming SSE-based job progress, session persistence via LocalStorage, internationalization via i18next, service worker updates for PWA support, and graph visualization. Its chat subsystem includes virtual-scrolled conversation lists, per-session history with date-based grouping, @mention page references, slash commands, and toolbars for summarization, skill invocation, and MCP integration. As a composite entity, `LLMWikiViewer` coordinates all major frontend pages — `ChatPage`, `SearchPage`, `UploadPage`, `GraphPage`, `HomePage` — under a shared `RootLayout` that provides consistent header, sidebar, alert banners, toast notifications, and animated page transitions. It is the central application surface through which users manage, query, and augment their personal wiki knowledge base with the assistance of large language models.
