## Overview

**Meta Platforms, Inc.**（原名 Facebook, Inc.）是一家全球性的科技巨头，专注于社交媒体、虚拟现实（VR）和人工智能研究。Meta 拥有 Facebook、Instagram、WhatsApp 等产品，并在 AI 领域投入大量资源，开发了包括 LLaMA 系列大语言模型在内的开源模型。

## Key Contributions

- **Hermes JavaScript 引擎** — 专为 React Native 优化的 JS 引擎，采用 AOT 编译和紧凑字节码
- **React Native** — 跨平台移动应用开发框架，Hermes 为其默认 JS 引擎
- **LLaMA** — 开源大语言模型系列，推动了开放权重模型的发展
- **PyTorch** — 主流深度学习框架，广泛用于 AI 研究
