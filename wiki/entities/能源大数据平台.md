# 能源大数据平台

**Energy Big Data Platform** — 王信杰参与的项目，为能源企业提供综合管理解决方案，通过调度算法优化资源配置。采用Spring Boot + MySQL + [[Redis]] + MyBatis-Plus + Spring Security + OAuth2 + Spring Cloud技术栈，支持5000 QPS并发，响应时间<500ms，系统可用性99.99%。
