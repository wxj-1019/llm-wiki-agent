# Anthropic

Anthropic is an AI safety company behind the [[Claude]] model family. In 2024, they released the Claude 3 series (Haiku, Sonnet, Opus) in March, followed by Claude 3.5 Sonnet in June — which remained [[SimonWillison]]'s favorite model through the end of the year. They introduced [[Claude Artifacts]], a groundbreaking feature for prompt-driven app generation. They also introduced Computer Use capability.

## Key developments in 2024

- [[Claude|Claude 3 Opus]] — GPT-4-beating model (March 2024)
- [[Claude|Claude 3.5 Sonnet]] — Willison's favorite model for 6+ months (June 2024)
- [[Claude Artifacts]] — interactive app generation within Claude interface
- Computer Use capability (October 2024)
- CLAUDE.md system for Claude Code projects
