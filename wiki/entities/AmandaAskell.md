# Amanda Askell

Amanda Askell is a researcher at [[Anthropic]], responsible for much of the work behind [[Claude]]'s character. She is known for advocating test-driven development for system prompts: "You don't write down a system prompt and find ways to test it. You write down tests and find a system prompt that passes them."
