# generateFromKnowledge

`generateFromKnowledge` is a feature panel within the **ChatPage** interface of the **LLMWikiViewer** that enables users to create structured **Skill** and **MCP** packages directly from the wiki's knowledge base. Rather than composing these packages manually, users can invoke this functionality to have the system synthesize relevant information from wiki pages, session context, and chat history into a coherent, production-ready output. The panel serves as a bridge between the conversational and analytical capabilities of the chat interface and the actionable deliverables of knowledge packages. It is typically accessed via the chat layout controls and is closely associated with the **@mention** and **slash command** systems, which allow users to specify source material and generation parameters. By automating the assembly of technical packages, `generateFromKnowledge` transforms the wiki from a passive repository into an active knowledge generation engine.
