PWA (Progressive Web App) is a web application delivery pattern that uses modern web capabilities to provide an app-like experience. In the [[LLMWikiViewer]] frontend, PWA support is implemented via `vite-plugin-pwa` with Workbox caching, enabling offline browsing of the wiki.

## Related Hooks
- [[useNetworkStatus]] — tracks connectivity for offline-aware UI
- [[usePWAInstall]] — prompts users to install the PWA
- [[useSWUpdate]] — notifies users when a new service worker is available

## Implementation
- Service worker generated by Vite PWA plugin
- Workbox strategies for runtime caching of wiki pages and static assets
- Manifest file defines app name, icons, and theme colors

See [[header-component-wiki-viewer-navigation-bar|Header Component]] and [[root-layout-main-application-layout-component|RootLayout Component]] for PWA integration details.
