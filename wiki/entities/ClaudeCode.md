# Claude Code

**Claude Code** 是 [[Anthropic]] 开发的 AI 编程助手 CLI 工具，基于 [[Claude]] 模型。[[Latte]] 项目是其一个可构建分支（fork），版本号 2.1.90。

## Key Facts

- 上游项目地址：https://docs.anthropic.com/en/docs/claude-code
- 核心功能包括代码生成、文件编辑、命令执行、代码搜索等
- [[Latte]] 分支移除了遥测和安全硬编码守卫，解锁了实验性功能
- 支持多模型接入（通过 OpenAI 兼容适配层）

## Connections

- [[Anthropic]] — 开发者
- [[Claude]] — 使用的模型
- [[Latte]] — 基于此项目的可构建分支
