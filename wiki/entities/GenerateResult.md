**GenerateResult** is a TypeScript type representing the output of the [[generateFromKnowledge]] function. It contains generated code or content for [[Skill]] (Jinja2 templates) or [[MCP]] server packages, produced from wiki knowledge via the LLM. Displayed in the [[ChatPage]] generate panel with preview, edit, and download options.

Related: [[generateFromKnowledge]], [[Skill]], [[MCP]]
