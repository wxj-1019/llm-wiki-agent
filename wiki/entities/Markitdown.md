# Markitdown

Microsoft's `markitdown[all]` library is the core auto-conversion engine used by the [[IngestTool|ingest tool]] to convert 20+ non-Markdown file formats (PDF, DOCX, PPTX, XLSX, HTML, JSON, XML, RST, RTF, EPUB, IPYNB, YAML, CSV, TSV, WAV, MP3, etc.) into Markdown before ingestion into the LLM Wiki.
