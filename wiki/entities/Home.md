# Home

In the context of this wiki, **Home** refers to the conceptual landing page or starting point of the application, typically navigated to via a hyperlink or icon button that returns the user to the root route. Within the `ErrorBoundary` component, a **Home** icon (imported from the Lucide React icon library) is rendered as part of the fallback user interface, providing users with a clear and accessible way to navigate back to the application's primary entry point after encountering a route-level or general error. This action is complemented by a **Refresh** option (using the `RefreshCcw` icon), offering users two recovery paths: returning to safety or retrying the current view. The presence of the Home link underscores its role as a stable, predictable fallback destination in the user experience, reinforcing navigational resilience.
