[[IGraph]] is a high-performance graph library available for Python (as `python-igraph`). In the [[BuildGraphTool|Build Graph Tool]], it is an optional dependency that enables the [[Leiden]] community detection algorithm via `leidenalg`. Imported as `ig`.

## Connections
- [[Leiden]] — community detection algorithm using igraph
- [[NetworkX]] — alternative graph library with Louvain support
- [[BuildGraphTool]] — optional dependency for advanced community detection
