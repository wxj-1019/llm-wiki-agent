# formatDistanceToNow

`formatDistanceToNow` is a utility function imported from `@/lib/dateUtils` used by [[NotificationDropdown]] and possibly other components to render relative timestamps (e.g., "5 minutes ago") in human-readable format.

## Usage
Used in the notification list to display how long ago each notification was created.
