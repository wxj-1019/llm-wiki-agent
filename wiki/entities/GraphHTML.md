[[GraphHTML]] is the interactive HTML visualization file generated by the [[BuildGraphTool|Build Graph Tool]]. Located at `graph/graph.html`, it is a self-contained page using the vis.js JavaScript library (loaded via CDN) to render the wiki knowledge graph with node physics, community-based coloring, and zoom/pan controls. No server is required to view it — simply open the file in a browser.

## Connections
- [[BuildGraphTool]] — generates this file
- [[VisJS]] — the visualization library used
- [[GraphJSON]] — the data source from which the HTML is generated
