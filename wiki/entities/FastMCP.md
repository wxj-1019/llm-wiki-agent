# FastMCP

FastMCP is a lightweight MCP server framework used by the [[WikiMCPServer|wiki-mcp-server]] to expose wiki resources, tools, and prompts via the stdio transport. It is installed via `pip install mcp>=1.2.0`.

## Connections

- [[MCP]] — the underlying protocol
- [[WikiMCPServer|wiki-mcp-server]] — the server that uses FastMCP
