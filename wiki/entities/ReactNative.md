## Overview

**React Native** 是 Meta（Facebook）开发的开源跨平台移动应用开发框架，允许开发者使用 JavaScript 和 React 构建原生 iOS 和 Android 应用。自 React Native 0.69.0 起，[[HermesJSEngine]] 成为其默认捆绑的 JavaScript 引擎，显著改善了应用启动性能和内存占用。

## Core Features

- **跨平台开发** — 一套代码库同时运行在 iOS 和 Android
- **原生性能** — 通过桥接层将 JavaScript 代码转换为原生组件
- **热重载** — 开发过程中实时预览代码变更
- **Hermes 集成** — 默认使用 Hermes 引擎，AOT 编译为紧凑字节码

## Connections

- [[HermesJSEngine]] — 默认 JavaScript 引擎，由 Meta 开发
- [[Meta]] — React Native 的开发和维护公司
