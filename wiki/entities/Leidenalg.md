[[Leidenalg]] is the Python package implementing the [[Leiden]] community detection algorithm. Used in conjunction with `python-igraph` (imported as `ig`) in the [[BuildGraphTool|Build Graph Tool]]. Activated via the `--leiden` CLI flag.

- [[Leiden]] — the algorithm this package implements
