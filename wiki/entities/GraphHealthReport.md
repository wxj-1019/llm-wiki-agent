[[GraphHealthReport]] is a structured report generated by the [[BuildGraphTool|Build Graph Tool]] via the `--report` flag. It includes: edge/node ratio analysis, orphan node percentage, community count, link density, a list of orphan nodes, god nodes (hub pages with degree > μ+2σ), fragile bridges (community pairs connected by only 1 edge), and phantom hubs (missing pages referenced by 2+ existing wiki pages). The report can be saved to `graph/graph-report.md` with `--save`.

## Connections
- [[BuildGraphTool]] — generates this report
- [[OrphanNodes]] — nodes with zero graph connections
- [[GodNodes]] — disproportionately connected hub nodes
- [[FragileBridges]] — single-edge community connections
- [[PhantomHubs]] — missing pages referenced by existing wikilinks
