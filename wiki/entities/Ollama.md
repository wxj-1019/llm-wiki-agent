**Ollama** is a local LLM inference server. Used by [[PgSearchBackend]] for generating text embeddings via `nomic-embed-text` model for semantic/vector search.

## Connections
- [[PgSearchBackend]] — uses Ollama for batch embedding generation
- [[pgvector]] — stores embeddings generated by Ollama
- [[nomic-embed-text]] — the embedding model
