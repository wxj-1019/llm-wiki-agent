# User Preferences

> 用户画像——偏好、沟通风格、期望。在 ingest/query 时注入作为系统上下文。
> 容量限制：~1375 字符。满时需整合或删除旧条目。

## 语言偏好

- 用户主要使用中文交流
- Wiki 内容可中英文混合
- Agent 回复使用中文

## 内容偏好

- 用户喜欢简洁精准的摘要，不喜欢冗长引用
- 关注知识之间的关联和交叉引用
- 重视矛盾检测和信息准确性

## 工作流偏好

- 倾向于批量处理多个文件
- 关注 wiki 的结构完整性和健康度
- 定期使用 health/lint 检查
