# Synthetic Data

Synthetic data refers to AI-generated training data deliberately created to train or fine-tune other AI models. In 2024, this became a deliberate advantage rather than a cheap substitute, contradicting [[ModelCollapse]] concerns about recursive training on AI-generated content.

## Key Advantages (per Phi-4 technical report)

- **Structured and Gradual Learning**: Each token generated by a language model is by definition predicted by preceding tokens, making reasoning patterns easier to follow
- **Curriculum Design**: Can structure complexity progression deliberately
- **Knowledge Distillation**: Larger models generate training data for smaller, cheaper models

## Notable Examples in 2024

- [[DeepSeekV3]] used reasoning data created by [[DeepSeekR1]]
- [[LLaMA|Llama 3.3 70B]] fine-tuning used 25M+ synthetically generated examples
- [[Microsoft|Phi-4]] emphasized synthetic data as a deliberate design choice

## Connection to Model Collapse

Contrary to the concern that models degenerate when trained on AI-generated content ([[ModelCollapse]]), synthetic data is now deliberately curated and provides structured learning advantages impossible with raw web scrapes.
