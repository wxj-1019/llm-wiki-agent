**RegexBasedYamlParsing** is a lightweight YAML parsing approach used by [[useConfigStore]]'s `parseGithubYaml`, `parseRssYaml`, and `parseArxivYaml` functions. Instead of using a full YAML parser library, these functions use line-by-line regex matching with simple patterns like `/name:\s*"(.+)"/` and `/label:\s*"(.+)"/` to extract structured data from config files. This is suitable when the YAML format is tightly controlled (auto-generated by the corresponding `build*Yaml` functions) and avoids an additional dependency.
