## Summary

GitHub 上名为 "Hermes" 的项目横跨多个领域：最热门的是 NousResearch 开源的 **Hermes Agent**（⭐128,747），一个自我进化的 AI Agent 框架；其次是 Meta/Facebook 的 **Hermes JavaScript 引擎**（⭐11,017），专为 React Native 优化。此外还有 Hermes WebUI、Hermes Workspace 等生态项目，以及 HashiCorp 的文档管理系统和 Cornell 的 RAG 优化框架等同名项目。

## Key Claims

- Hermes Agent 是目前 GitHub 上星标数最高的 AI Agent 项目之一（128K+ stars），由 Nous Research 开发
- Hermes Agent 的核心理念是"自我改进闭环"：从经验中创建技能、使用中改进技能、跨会话搜索历史、构建用户模型
- Hermes JS 引擎是 React Native 的默认 JavaScript 引擎，采用 AOT 编译和紧凑字节码优化启动性能
- Hermes 生态系统包含 WebUI、Workspace、Self-Evolution、Orange Book 等多个子项目

## Key Quotes
