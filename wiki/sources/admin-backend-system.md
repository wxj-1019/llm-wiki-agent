## Summary
后台管理系统是平台的运维与运营管理中枢，独立于前台 API 服务（端口 8585）运行在端口 8586。系统提供完整的系统监控、数据管理、用户管理、AI 配置、审计日志等功能，并采用独立的审计中间件确保管理操作全程留痕。

## Key Claims
- 后台服务独立部署（端口 8586），与前台服务（端口 8585）分离，避免管理操作影响用户体验
- 所有管理操作通过审计中间件自动记录，满足合规要求
- 提供系统/数据库/Redis 三层监控，覆盖 CPU、内存、磁盘、数据库连接、慢查询、表空间、Redis 内存与命中率等
- 主力选股管理支持选股记录查询、统计分析、模板管理、手动触发选股、回测执行、策略版本管理与用户收藏管理
- 数据质量模块自动检查股票数据完整性与一致性
