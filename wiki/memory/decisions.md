# Agent Decisions Log

跨会话持久决策记录。每条决策包含上下文、理由和后续验证状态。

## Decision Format

```yaml
---
decision_id: D-YYYYMMDD-NNN
date: YYYY-MM-DD
session_id: S-YYYYMMDD-NNN
topic: "简要描述"
status: active | superseded | validated
superseded_by: D-YYYYMMDD-NNN  # 可选
---
```
